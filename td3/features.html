Images et Métadonnées

1 - Modes d'Utilisation

L'application propose deux modes d'utilisation :

  • Mode CLI (Ligne de Commande) : Toutes les fonctionnalités sont accessibles via des commandes. Ce mode est idéal pour une utilisation rapide et efficace, directement dans le terminal.
  • Mode GUI (Interface Graphique) : Les mêmes fonctionnalités sont disponibles dans une interface visuelle conviviale. En plus, ce mode permet d’afficher les images, quelle que soit leur taille, grâce à un affichage adapté avec défilement (via JScrollPane).

2 - Liste des Fonctionnalités Principales

Lister les fichiers images d’un répertoire.
Affiche tous les fichiers image présents dans un dossier spécifié.
Extraire les statistiques générales d’un répertoire :
Nombre total de fichiers :
Affiche le nombre total de fichiers dans le répertoire.
Nombre d’images :
Compte le nombre total d’images.
Nombre d’images par format :
Indique la répartition des images par format (PNG, JPEG, WEBP).
Extraire les statistiques d’un fichier particulier :
Type MIME :
Affiche le type MIME du fichier.
Taille du fichier :
Indique la taille en kilo-octets (KB).
Date de dernière modification :
Affiche la date de dernière modification du fichier.
Afficher les métadonnées EXIF et XMP d’une image :
Dimensions de l’image :
Affiche la largeur et la hauteur en pixels.
Résolution (DPI) :
Indique la résolution de l’image en points par pouce.
Position GPS (GeoTag) :
Affiche les coordonnées GPS si disponibles.
Permettre la recherche d’images :
Nom ou partie du nom :
Recherche les images en fonction du nom.
